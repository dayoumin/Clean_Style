1. Executive Summary

핵심 결론 4가지:
  1. DIT-2가 국제적으로 가장 널리 쓰이나, 센터 채점 서비스 종속이 실무 적용의 큰 제약
  2. MJT/MCT는 태도가 아닌 '역량(competence)'을 구조적으로 산출 — 일반 리커트 척도와 용도가 다름
  3. 공직 맥락의 딜레마에는 MES(시나리오 기반) 또는 국내 공직윤리/청렴 동기 척도가 보완적으로 유용
  4. Rest 4요소(민감성–판단–동기–실행) 단위 측정은 EDM-CS로 가능하나, 공공기관 표본 재타당화 필수

2. 국제 표준 EDM · 도덕 판단 도구

MJT / MCT 국제 표준
정식 명칭
Moral Judgment Test / Moral Competence Test
개발자
Georg Lind (1970s~)
측정 대상
10세 이상 (아동~성인, 직역 무관)
핵심 지표
C-index (C-score) — 논증의 도덕적 질에 근거한 판단 '역량'
문항 구조
2개 딜레마 × 찬반 논증 평가 = 26개 반응항목
척도: −4(매우 반대) ~ +4(매우 찬성)
신뢰도
Test-retest r=0.90 / 한국어판 α=0.669
한국어판
박건영(2014) 표준화 · 타당화 연구 존재
접근성
저자 무료 제공, 개인 선발/진단 목적 사용 불가
핵심: 태도 자기보고가 아닌 '반대 논증 평가 과제'를 통한 역량 산출. 청렴교육 전후 변화 평가에 강점. 채점/해석이 구조적이며 일반 합산점수와 다름.
DIT-2 국제 표준
정식 명칭
Defining Issues Test, Version 2
개발자
Rest, Narvaez, Thoma, Bebeau (1999)
핵심 지표
3개 발달적 스키마: 개인적 이익 / 규범 유지 / 후기-관습(원리 중심)
문항 구조
5개 딜레마 × 고려사항 중요도 평가(rating) + 순위(ranking)
신뢰도
N2 α=0.81 (스토리 단위), DIT1-P α=0.76
한국어판
KDIT(한국판) 사용 사례 보고 (간호대학생 등)
접근성
센터 채점 서비스 종속 (자체 채점 불가)
핵심: 도덕 추론 발달을 스키마 관점에서 측정하는 대표 도구. 국제 비교/발달 수준 비교에 최적. 단, 채점 서비스 비용/일정/데이터 보안을 사전에 확정해야 함.
MES 국제 표준
정식 명칭
Multidimensional Ethics Scale
개발자
Reidenbach & Robin (1990)
핵심 차원
Moral Equity (정의/공정) · Relativism (문화적 수용) · Contractualism (계약 위반)
문항 구조
시나리오당 8개 의미분별(semantic differential)
예: Just/Unjust, Fair/Unfair, Morally Right/Not Right
신뢰도
α=0.92 (McMahon, 2002)
한국어판
세무전문가/세무대리인 집단에서 활용 사례
접근성
문항 학술문헌에 널리 인용, 재배포 시 출처 표기/저작권 확인 필요
핵심: 공공기관 부패/이해충돌/특혜 시나리오를 설계하면 다차원 윤리 판단의 근거를 분해 가능. 요인구조 재현성 비판이 존재하므로 CFA/측정불변성 재검증 권고.
EDM-CS 2025 신규
정식 명칭
Ethical Decision-Making Competence Scale
개발자
2025년 간호교육 맥락 개발
핵심 차원
4요인: 민감성 · 판단 · 동기 · 실행 (Rest 4요소)
문항 구조
25문항 (4요인) / 2차 요인모형 18문항
신뢰도
전체 α=0.90, 하위요인 α=0.73~0.80
한국어판
미확인 (터키어 버전 타당화만 보고)
접근성
학술지 저작권 범위, 공공기관 표본 재타당화 필수
핵심: 윤리적 의사결정을 구성요소(민감성-판단-동기-실행) 단위로 분해. 교육/개입 프로그램과 연결 용이. 현재 간호대학생 표본이므로 공공기관 이식 시 재타당화 필요.

3. 한국 공공부문 맥락 도구

동기론적 공직윤리 척도 한국 개발
개발자
김상우 (2023, 상명대 박사학위논문)
측정 대상
중앙행정기관 소속 공무원 (본조사 N=128)
핵심 차원
4요인: 소극적 성실 · 적극적 성실 · 책임 · 청렴
문항 구조
예비 44문항 → 내용타당도 후 35문항 (EFA→CFA→고차CFA)
신뢰도
초록 수준에서 α/CR/AVE 수치 미제시 (원문 확인 필요)
접근성
RISS 저작권 보호 명시, 사용 허가/재검증 이슈
핵심: 부패 방지를 넘어 '성실/책임/청렴의 윤리적 동기 내면화'를 다차원 측정. 공기업/준정부 일반화에는 재타당화 필요.
공직가치 모형 기반 측정 한국 개발
개발자
김상묵 (2017)
측정 대상
국가공무원 648명 설문조사
핵심 차원
5유형: 윤리적 · 민주적 · 혁신적 · 전문직업적 · 전통적
문항/신뢰도
초록 수준에서 문항수/α 미제시 (원문 확인 필요)
접근성
학술지 접근 정책에 따름
핵심: 청렴을 넘어 공직 가치 전체(윤리/민주/혁신/전문/전통)를 인사관리 프레임으로 확장. 행동지표(부패 딜레마/신고의도)와는 별도 연결 설계 필요.
KIPF 실험설문 도구 한국 개발
출처
한국조세재정연구원 정책연구 보고서 21-18 (2021)
저자
하세정 · 박한준 · 조윤직
측정 대상
공공기관 직원의 윤리의식/윤리적 의사결정 행태
핵심 변수
윤리의식, 부패행위 선택/내부고발, 공공봉사동기, 조직정치, 부패방지교육, 처벌 적정성, 감사 빈도, 처리과정 신뢰성
주요 결과
"민간기업 대비 공공기관이 전반적으로 더 윤리적인 의사결정"
접근성
공공누리 4유형 (출처표시+상업적 이용 금지+변경금지)
핵심: 공공기관 핵심 관심(부패 참여/신고)을 직접 측정. 정책변수와 직접 연결 가능. '표준화 척도'인지 '보고서 한정 실험설문'인지 확인 후 재사용 절차 결정 필요.

공공기관 설문 설계 선택 가이드

"윤리교육의 전후 변화를 엄밀히 보고 싶다"
→ MJT/MCT — 프로그램 평가에 민감. 단, 선발/진단 목적 제한.
"국제 비교 · 발달 수준 비교가 필요하다"
→ DIT-2 — 검증 축적이 강점. 단, 센터 채점 서비스 종속이 운영 리스크.
"공공부문 부패 · 공정 딜레마에서의 판단과 행동의도를 직접 보고 싶다"
→ MES (공공부문 시나리오 구성) 또는 KIPF 방식 실험설문. 요인구조 재검증 병행.
"청렴 · 책임 · 성실의 동기/가치 지향을 인사관리와 연결하고 싶다"
→ 동기론적 공직윤리 척도 또는 공직가치 모형. 문항 재사용/저작권 · 기관 유형별 재검증 필수.
설계 원칙: (i) 도덕 판단의 발달/역량, (ii) 업무 맥락의 딜레마 판단, (iii) 청렴/성실/책임의 내면화(가치/동기)를 구분하여 설계하면 결과 해석이 안정적이다.

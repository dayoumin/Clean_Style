PWA 추가 작업 계획

manifest.json + 아이콘만으로 "홈 화면에 추가" 구현. Service Worker는 나중에.

1. 아이콘 제작

필요 파일

파일크기용도

icon-512.png512 x 512Android 스플래시, 설치 프롬프트
apple-touch-icon.png180 x 180iOS 홈 화면
favicon.ico32 x 32브라우저 탭
maskable-icon-512.png512 x 512Android 적응형 (safe zone 80%)

AI 이미지 생성 프롬프트

DALL-E / Midjourney Prompt

Minimal flat app icon for an integrity self-assessment tool.
Purple gradient background (#7C5CFC to #6B4CE0).
White shield silhouette in the center with a small checkmark inside.
No text, no border, no shadow. Clean and modern.
Square with rounded corners (20% radius).
Output: 1024x1024 PNG with transparent-safe edges.

변환 작업

1024 x 1024 원본에서 리사이즈:

512, 192, 180, 32px 각각 생성

maskable 버전: 중앙 80% 안전 영역에 아이콘, 배경색 꽉 채움

도구: Squoosh, sharp, 또는 npx pwa-asset-generator

2. 파일 생성

public/manifest.json

{
  "name": "나의 청렴 스타일은?",
  "short_name": "청렴스타일",
  "description": "3분 자기발견 테스트",
  "start_url": "/",
  "display": "standalone",
  "background_color": "#F8F7FF",
  "theme_color": "#7C5CFC",
  "icons": [
    { "src": "/icon-192.png", "sizes": "192x192", "type": "image/png" },
    { "src": "/icon-512.png", "sizes": "512x512", "type": "image/png" },
    { "src": "/maskable-icon-512.png", "sizes": "512x512",
      "type": "image/png", "purpose": "maskable" }
  ]
}

src/app/layout.tsx 메타태그

<link rel="manifest" href="/manifest.json" />
<meta name="theme-color" content="#7C5CFC" />
<link rel="apple-touch-icon" href="/apple-touch-icon.png" />
<meta name="apple-mobile-web-app-capable" content="yes" />
<meta name="apple-mobile-web-app-status-bar-style" content="black-translucent" />

3. 작업 순서

1 아이콘 원본 생성 (AI 또는 수작업) ~10분
2 리사이즈 + maskable 변환 ~5분
3 public/ 폴더 생성 + 아이콘 배치 ~2분
4 manifest.json 작성 ~3분
5 layout.tsx 메타태그 추가 ~3분
6 favicon.ico 추가 ~2분
7 로컬 테스트 (DevTools > Application > Manifest) ~5분

4. Service Worker

지금은 생략. manifest.json + 아이콘만으로 "홈 화면에 추가"는 동작합니다.
오프라인 캐싱이 필요해지면 next-pwa 또는 serwist 추가.

5. 검증 체크리스트

  • Chrome DevTools > Application > Manifest 정상 표시
  • 모바일 Chrome "홈 화면에 추가" 동작
  • iOS Safari "홈 화면에 추가" 시 아이콘 표시
  • 설치 후 standalone 모드 (브라우저 UI 없음)
  • 테마 색상 상태바 반영

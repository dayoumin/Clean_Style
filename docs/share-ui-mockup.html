시안 비교 정리

시안 A — 하단 3버튼

  • 장점: 한 줄에 모든 액션, 공간 절약
  • 단점: 버튼 3개가 좁을 수 있음 (작은 화면)

시안 B — 별도 줄 분리

  • 장점: 기존 레이아웃 변경 최소화, 링크 복사가 눈에 잘 띔
  • 단점: 세로 공간을 약간 더 차지

시안 C — 카드 아래 공유 바

  • 장점: 결과 확인 직후 바로 공유 가능, 자연스러운 흐름
  • 단점: 스크롤 없이 보이는 영역이 줄어듦

  • 상단에 "다른 사람의 결과입니다" 배너 표시

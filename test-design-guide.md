## DISPLAY_MAX=5 결정 근거

15문항 구조에서 축당 평균 2~4점. MAX=5일 때 차트 40~80% 채움으로 적정.
MAX=8이면 25~50%로 허전. 동적 스케일은 응답자 간 비교 불가.
문항 24개 이상으로 늘리면 MAX 상향 가능.

### 정규화 방식

점수는 축별 이론상 최대로 나눠 0~DISPLAY_MAX(5) 범위로 정규화:

```
normalized = round((raw / axisMax) * DISPLAY_MAX)
```

- `axisMax`는 `AXIS_MAXIMUMS`로 문항 구조에서 자동 계산 (축당 12~15)
- 정규화된 값이 차트 영역, 라벨, 툴팁 모두에 동일하게 사용됨
- 5점 = 해당 축에서 선택 가능한 모든 문항을 해당 방향으로 고른 경우

## 3축 설계의 이론적 근거와 한계

### 참고한 개념

| 축 | 참고 이론/개념 | 출처 | 심층 근거 |
|---|---|---|---|
| 원칙↔유연 | 의무론 vs 결과론, 윤리풍토 Rules/Independence | Kohlberg, Forsyth EPQ, Schwartz, HEXACO C/A | `research/axis2_principle_flexibility.md` |
| 투명↔신중 | 정직 vs 신중성 독립 차원, 투명성 역설 | VIA, HEXACO H/C, ALQ, DCA 척도 | `research/rese.md` |
| 독립↔협력 | Agency ↔ Communion 직교 구조 | 대인관계 원형모델, 자기해석 이론, GLOBE | `research/axis3_independence_cooperation.md` |

### 설계 방식

- 위 이론에서 **개념만 차용**하여 공공 연구기관 맥락에 맞게 재구성
- 학술적으로 검증된 표준 척도(validated scale)를 사용한 것이 아님
- 문항은 공공기관 실무 상황(연구비, 계약, 외부 협력)을 기반으로 자체 개발

### 검증 상태

| 항목 | 상태 | 비고 |
|---|---|---|
| 내용 타당도 (Content Validity) | 부분 완료 | 문항 매핑 검토 + 사회적 바람직성 매칭 반영 |
| 구성 타당도 (Construct Validity) | 미검증 | 요인분석 등 통계적 검증 미실시 |
| 신뢰도 (Reliability) | 미검증 | Cronbach's alpha 등 미측정 |
| 파일럿 테스트 | 미실시 | 배포 후 진행 예정 |
| 외적 타당도 | 해당 없음 | 자기발견 도구 목적, 인사 평가 용도 아님 |

### 포지셔닝

이 테스트는 **"과학적 진단 도구"가 아닌 "자기발견 및 교육 보조 도구"**로 설계됨.

- 16Personalities(MBTI 기반), StrengthsFinder 등 대중적 성향 테스트와 유사한 포지셔닝
- 결과 화면에서 "공식 평가와 무관", "부담 없이 즐기는 셀프 테스트" 명시
- AI 조언도 "옳고 그름 판단 금지" 원칙으로 프레이밍

### 알려진 한계

1. **투명↔신중 축 잔여 불균형**: 투명 17회 vs 신중 14회 (차이 3, 허용 범위). 완전 균형은 Q16~Q18 추가로 달성 예정
2. **원칙-독립 공동출현**: 9문항에서 principle+1과 independence+1이 동시 부여 → 독립 측정 어려움
3. **문항 수 제한**: 15문항으로 3축을 측정하면 축당 문항 수가 적어 변별력 한계
4. **응답 편향**: 강제선택형이지만, 사회적 바람직성 효과를 완전히 제거하기 어려움
